# 💰 MoneyMap – Personal Finance Dashboard

## 📌 Overview
MoneyMap is a simple and interactive finance dashboard built using React.  
It helps users track their daily expenses, manage transactions, and understand spending behavior through a clean and user-friendly interface.

---

## 🚀 Features

- 📊 Dashboard Overview (Total Balance, Income, Expenses)
- 🧾 Transaction History with Search & Filter
- 🔐 Role-Based Access  
  - Admin → can add transactions  
  - Viewer → read-only access  
- 📈 Spending Insights (total expense + category analysis)
- 🌙 Dark Mode Toggle
- 💾 Data Persistence using Local Storage (data remains after refresh)
- 📥 Export Data Feature (download transactions as JSON)
- ✨ Smooth UI interactions (hover effects and transitions)
- 📅 Date Range Filtering
- 🏷️ Category Filtering
- 🗑️ Delete Transactions (Admin Only)
- 📊 CSV Export for Power BI Integration
- 📈 Monthly Summary Dashboard
- 📉 Expense Bar Chart Visualization

## 💡 My Approach

I structured the project using reusable React components such as:
- Overview Panel
- Transaction List
- Role Toggle
- Spending Insights  

State is managed using `useState`, and `useEffect` is used for storing data in local storage.  
I focused on keeping the UI simple, interactive, and easy to use.

## 📊 Power BI Integration

The transaction data can be exported as CSV and imported into Power BI for:

- KPI Analysis
- Income vs Expense Tracking
- Category-wise Spending Analysis
- Monthly Trends
- Interactive Slicers and Filters


## ▶️ How to Run

1. Navigate to project folder  
   cd finance-Dashboard  

2. Install dependencies  
   npm install  

3. Start development server  
   npm run dev  

4. Open in browser  
   http://localhost:5173/
